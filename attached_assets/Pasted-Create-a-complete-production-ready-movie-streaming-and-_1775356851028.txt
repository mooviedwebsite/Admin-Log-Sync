Create a complete production-ready movie streaming and downloading website named **"MOOVIED"** with a modern Netflix-style UI/UX.

### 🔥 Core Requirements

* Frontend must be **pure HTML, CSS (TailwindCSS), and Vanilla JavaScript**
* Must be fully deployable on **GitHub Pages (no backend server)**
* Use **Firebase (Authentication + Firestore + Storage)** as the only backend
* No Node.js, no Express, no Vercel, no server-side code

---

### 🎨 UI / UX Design

* Netflix-inspired premium design
* Color palette:

  * Black (#000000)
  * White (#FFFFFF)
  * Yellow (accent)
* Use **glassmorphism effects** (blur, transparency)
* Smooth animations and hover effects
* Fully responsive (mobile + desktop)
* Modern fonts and spacing

---

### 🏠 Pages / Features

#### 1. Home Page

* Featured movie banner (auto slider)
* Trending movies section
* Latest uploads section
* Movie cards (poster, title, rating, year)
* Hover preview effect (like Netflix)

#### 2. Movie Details Page

* Poster, title, description
* Genre, duration, release year
* Video player (streaming using Firebase Storage links)
* Download button
* Subtitle support (upload + select)

#### 3. Authentication System (Firebase Auth)

* Sign up / Login
* Email + Password
* Store user data in Firestore
* User profile system

#### 4. User Features

* Bookmark / Favorites system
* Watch history
* Profile page
